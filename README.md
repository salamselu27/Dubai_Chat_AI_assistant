# ✈️ Dubai Trip AI Assistant
An AI-powered personalized travel planner that generates custom itineraries using GPT-3.

## 🚀 Overview
This project is an interactive chatbot designed to help users plan trips to Dubai by providing personalized recommendations based on user input. It focuses on delivering an intuitive user experience and high-quality travel guidance.

## 🛠️ Tech Stack
* **Language:** Python
* **Framework:** Streamlit (Interactive UI)
* **AI Model:** OpenAI GPT-3 API

## ✨ Key Features
* **Real-Time Recommendations:** Generates instant travel suggestions through an AI chatbot.
* **Personalized Itineraries:** Creates daily plans based on specific user preferences.
* **Interactive UI:** Built with Streamlit for a seamless, responsive experience.
* **API Integration:** Successfully integrated OpenAI's GPT-3 to handle complex natural language queries.

## ⚙️ Setup & Installation
1. Clone the repo: `git clone https://github.com/salamselu27/Dubai_Chat_AI_assistant.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Add your OpenAI API Key to `.env`
4. Run the app: `streamlit run app.py`
